# 更新日志

## v2.4.0 (2026-04-26)

### 重大更新

#### 多模型 AI 支持
- 支持两种 API 协议：OpenAI 兼容格式 / Anthropic 格式
- 国内大模型开箱即用：小米 MiMo、DeepSeek、通义千问、硅基流动等
- 设置界面简化：仅需填写协议类型、API 地址、API Key、模型名称
- 每次选择服务商后可测试连接
- API Key 安全存储于用户数据目录，不随应用打包

#### 流式输出
- AI 对话采用 SSE 流式传输，文字逐字显示
- 问答、摘要、润色模式均支持流式输出
- 支持中途取消（停止按钮）

#### 跨笔记知识检索 (RAG)
- 对话模式下自动检索知识库中最相关的笔记
- 基于 BM25 关键词匹配算法（轻量、无需外部依赖）
- 支持中英文混合分词
- 顶部开关可随时启用/禁用知识库检索

#### 文件菜单「设置」入口
- 文件菜单新增「设置」选项（快捷键 `Ctrl+,`）
- 自定义 About 弹窗（替代系统原生对话框）
- About 弹窗支持「查看文档」链接

### Bug 修复
- 修复 AI 流式对话无响应的竞态条件（requestId 在调用前生成）
- 修复 React 重复 key 警告（消息使用唯一 ID）
- 移除 `@anthropic-ai/sdk` 依赖（完全通过 IPC 调用替代）

## v2.3.0 (2026-04-26)

### 新增功能

#### Obsidian 语法解析器
- 新增 `src/utils/obsidianParser.ts` — 完整的 Obsidian Markdown 语法解析
- 支持 `[[双向链接]]` 和 `[[目标|别名]]` 语法解析
- 支持 `#标签` 和 `#标签/子标签` 提取
- 支持 `> [!note]` Callout 语法解析
- 支持 `![[嵌入]]` 嵌入语法识别
- 支持 Frontmatter YAML 元数据解析
- 提供 `obsidianToMarkdown()` 转换函数
- 提供 `extractPlainText()` 纯文本提取（用于 AI 分析）

#### 相关笔记推荐
- 新增 `src/components/RelatedNotes.tsx` — 智能相关笔记推荐
- 基于双向链接匹配（最高优先级）
- 基于共同标签匹配
- 基于内容关键词相似度匹配
- 编辑器工具栏新增「关联」按钮，可展开/收起相关笔记面板

#### Excel 导入增强
- Excel 导入现在保留 Markdown 表格结构（不再是纯文本）
- 支持多 Sheet 导入，每个 Sheet 独立显示
- 表格自动对齐，保留表头
- 显示数据行数统计

### 技术改进
- 修复 `sn` 参数隐式 any 类型
- TypeScript 编译检查通过

## v2.2.0 (2026-04-26)

### 新增功能

#### 可拖拽面板
- 新增 `ResizablePanel` 组件 — 面板边界可拖拽调整宽度
- 侧边栏、笔记列表、AI 面板均可自由调整
- 编辑器区域自动伸缩
- 各面板有最小/最大宽度限制

#### 笔记列表排序
- 支持按更新时间、标题、分类排序
- 点击切换升序/降序
- 当前排序状态高亮显示

#### 响应式布局
- 笔记列表宽度自适应
- 宽度 < 240px 时隐藏预览文字
- 宽度 < 200px 时隐藏标签和阅读时间

#### Markdown 实时预览
- 编辑器支持「编辑」和「实时预览」模式切换
- 默认进入实时预览模式
- 工具栏新增切换按钮

#### Frontmatter 元数据显示
- 自动解析 Frontmatter 中的 `synced_at` 和 `source_path`
- 以清晰格式显示在编辑器工具栏
- 原始 YAML 不再显示在正文中

#### 分类管理增强
- 右键菜单：重命名分类、隐藏分类
- 隐藏分类折叠区域，可展开恢复
- 导入文档时检测同名分类，提示合并或新建

### 界面优化

#### 弹窗统一优化
- 所有弹窗支持 ESC 键关闭
- 点击遮罩层可关闭弹窗
- 内容溢出时可滚动
- 底部统一关闭按钮

#### 深色主题统一
- 编辑器预览区颜色与整体主题一致
- 工具栏图标颜色统一
- 标题、正文、代码块颜色分层

#### 工具栏重构
- 精简为两行布局：标题+按钮 / 元信息
- 按钮改为轻量文字样式

### Bug 修复
- 修复 preload.js 路径引用错误
- 修复 SyncResult 类型定义与实际返回值不匹配
- 修复浅色模式下部分组件仍显示深色的问题
- 修复搜索放大镜点击无反应的问题
- 修复 Obsidian 同步按钮无反应的问题
