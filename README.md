# 知识库助手 - 使用说明

## 一、项目简介

知识库助手是一个基于本地存储的个人知识库系统，结合 AI 能力（Claude API）提升工作和生活效率。支持笔记管理、AI 问答、内容摘要、写作润色等功能。

**当前版本：v2.3.0**

## 二、环境要求

| 要求 | 说明 |
|------|------|
| 操作系统 | Windows 10/11 (64位) |
| Node.js | 18.x 或更高版本 |
| Claude API Key | 可从 anthropic.com 申请（可选） |

## 三、快速开始

### 步骤 1：安装依赖

```bash
# 进入项目目录
cd d:\cc-projects\projects\knowledge-base

# 安装项目依赖
npm install
```

### 步骤 2：配置 Claude API Key（可选）

1. 访问 https://console.anthropic.com/
2. 创建 API Key
3. 运行项目后，在设置界面中输入 API Key

### 步骤 3：启动应用

```bash
# 开发模式运行
npm run dev:electron
```

或构建后运行：

```bash
# 构建生产版本
npm run build

# 运行构建后的应用
npm run preview
```

## 四、主要功能

### 4.1 笔记管理

- **创建笔记**：点击笔记列表上方的 + 按钮
- **编辑笔记**：点击笔记列表中的任意笔记
- **删除笔记**：鼠标悬停在笔记上，显示删除按钮
- **自动保存**：编辑内容后 1 秒自动保存
- **批量操作**：按住 Shift 键多选笔记，支持批量删除和移动分类
- **Markdown 实时预览**：编辑器支持实时预览模式，所见即所得

### 4.2 笔记列表

- **排序功能**：支持按更新时间、标题、分类排序，点击切换升序/降序
- **响应式布局**：列表宽度可拖拽调整，内容随宽度自适应
- **搜索反馈**：搜索后显示匹配结果数量

### 4.3 分类管理

- 侧边栏可切换分类视图
- 支持按分类筛选笔记
- 支持批量移动笔记到其他分类
- **右键菜单**：右键分类可重命名或隐藏
- **隐藏分类**：不常用的分类可隐藏，需要时再显示
- **导入去重**：导入文档时检测同名分类，提示合并或新建

### 4.4 可拖拽面板

- 侧边栏、笔记列表、AI 面板均可通过拖拽分割线调整宽度
- 编辑器区域自动伸缩
- 各面板有最小/最大宽度限制

### 4.5 AI 功能

| 功能 | 说明 | 使用方法 |
|------|------|----------|
| 知识问答 | 基于笔记内容回答问题 | 打开笔记 → AI 面板 → 问答模式 |
| 内容摘要 | 自动总结笔记要点 | 打开笔记 → AI 面板 → 摘要模式 |
| 写作润色 | 改进笔记写作质量 | 打开笔记 → AI 面板 → 润色模式 |
| 自动标签 | AI 建议笔记标签 | 编辑器工具栏 → 标签按钮 |
| 相关笔记推荐 | 智能推荐关联笔记 | 编辑器工具栏 → 关联按钮 |

### 4.6 Obsidian 语法支持

完整解析 Obsidian Markdown 语法，同步的笔记保留原始结构：

| 语法 | 说明 |
|------|------|
| `[[双向链接]]` | 自动解析笔记关联关系 |
| `[[目标\|别名]]` | 支持别名显示 |
| `#标签` / `#标签/子标签` | 自动提取标签体系 |
| `> [!note]` Callout | 识别并高亮显示 |
| `![[嵌入]]` | 识别嵌入内容 |
| Frontmatter | 解析 YAML 元数据 |

### 4.7 相关笔记推荐

基于多种维度智能推荐关联笔记：
- **双向链接匹配**：优先推荐被当前笔记链接的笔记
- **共同标签匹配**：推荐具有相同标签的笔记
- **内容相似度**：基于关键词匹配推荐内容相关的笔记

### 4.6 Obsidian 同步

1. 打开设置 → Obsidian 同步
2. 选择 Obsidian Vault 路径
3. 启用自动同步（可选）
4. 配置排除文件夹
5. 欢迎页点击「同步笔记」一键同步

### 4.8 文档导入

支持导入以下格式：
- Markdown (.md)
- Word (.docx)
- **Excel (.xlsx, .xls)**：保留 Markdown 表格结构，支持多 Sheet
- PDF (.pdf)
- 纯文本 (.txt)

### 4.9 统计分析

- 总笔记数、本周/本月更新数
- 标签总数、总字数
- 知识来源分布
- 分类分布
- 更新趋势图

### 4.10 回收站

- 删除的笔记进入回收站，可恢复
- 支持清空回收站

### 4.11 导出功能

- 支持导出为 Markdown、HTML、TXT 格式
- 支持批量导出

## 七、常见问题

### Q1：运行报错 "app is not defined"

这通常是 Electron 环境问题。解决方法：

```bash
# 清除 node_modules
rm -rf node_modules

# 重新安装
npm install
```

### Q2：Claude API 调用失败

1. 确认 API Key 已正确配置
2. 检查网络连接
3. 查看 API Key 是否有足够配额

### Q3：笔记不显示

1. 确认笔记目录存在（默认：Documents/KnowledgeBase）
2. 检查笔记文件是否为 .md 格式
3. 查看应用日志了解具体错误

### Q4：Obsidian 同步提示"不是有效的 Vault"

确保选择的目录包含 `.obsidian` 文件夹，这是 Obsidian Vault 的标识。
