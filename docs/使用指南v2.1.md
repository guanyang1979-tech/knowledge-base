# 知识库助手使用指南 v2.1.0

## 快速开始

### 安装与启动

1. 双击运行 `知识库助手.exe`
2. 首次启动会显示启动封面，3 秒后进入主界面
3. 点击右上角齿轮图标配置 Claude API Key（可选，用于 AI 功能）

### 基本操作

#### 创建笔记
- 点击笔记列表上方的 `+` 按钮
- 或使用快捷键 `Ctrl+N`
- 输入标题，选择分类，点击创建

#### 编辑笔记
- 在笔记列表中点击要编辑的笔记
- 在右侧编辑器中修改内容
- 支持 Markdown 语法
- 笔记会自动保存（1 秒防抖）

#### 搜索笔记
- 在顶部搜索框输入关键词
- 支持按标题、内容、标签搜索
- 使用快捷键 `Ctrl+F` 快速聚焦搜索框

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建笔记 |
| `Ctrl+S` | 保存当前笔记 |
| `Ctrl+F` | 聚焦搜索框 |
| `Ctrl+Shift+F` | 全局搜索 |
| `Ctrl+Shift+D` | 切换侧边栏 |
| `Ctrl+E` | 导出当前笔记 |

---

## 批量操作

### 多选笔记
- 按住 `Shift` 键点击笔记，可多选
- 选中后会显示批量操作工具栏

### 批量删除
- 选中多篇笔记后，点击工具栏的"删除"按钮
- 确认后将选中笔记移到回收站

### 批量移动分类
- 选中多篇笔记后，在工具栏选择目标分类
- 点击"移动"按钮完成分类转移

---

## AI 功能

### 配置 API Key
1. 点击右上角齿轮图标打开设置
2. 输入 Claude API Key
3. 点击"测试"验证 Key 是否有效
4. 点击"保存"

### AI 对话
- 在右侧面板选择"对话"模式
- 输入问题或指令
- AI 会结合当前笔记内容回答

### AI 问答
- 选择"问答"模式
- 基于当前笔记内容回答问题

### AI 摘要
- 选择"摘要"模式
- 点击发送生成笔记摘要

### AI 润色
- 选择"润色"模式
- 点击发送改进笔记写作质量

### AI 建议标签
- 在编辑器工具栏点击"标签"按钮
- AI 会根据内容自动建议标签

---

## 笔记管理

### 分类管理
- 在左侧边栏选择分类
- 创建笔记时可选择或新建分类

### 标签管理
- 在笔记的 frontmatter 中添加 `tags: [标签1, 标签2]`
- 使用 AI 建议标签功能自动添加

### 回收站
- 删除的笔记会移到回收站
- 可在回收站中恢复或永久删除
- 支持清空回收站

---

## 导入导出

### 导入文档
- 支持格式：Markdown、Word、Excel、PDF
- 点击工具栏的导入按钮
- 选择文件和目标分类

### 导出笔记
- 支持格式：Markdown、HTML、TXT
- 使用快捷键 `Ctrl+E` 导出当前笔记
- 或在工具栏点击导出按钮

---

## Obsidian 同步

### 配置
1. 打开设置面板
2. 选择 Obsidian Vault 路径
3. 启用自动同步（可选）
4. 配置排除文件夹

### 同步方式
- 自动同步：启动时自动同步
- 手动同步：点击同步按钮

---

## 设置选项

### 基本设置
- **Claude API Key**：用于 AI 功能
- **笔记存储目录**：笔记保存位置
- **云同步目录**：iCloud/Google Drive 路径
- **外观**：浅色/深色主题

### 高级设置
- **排除文件夹**：不导入的文件夹列表
- **Obsidian 同步**：Vault 路径和自动同步设置

---

## 统计分析

点击工具栏的统计按钮查看：
- 总笔记数、本周/本月更新数
- 标签总数、总字数
- 知识来源分布（Obsidian/导入/手动）
- 分类分布
- 更新趋势（最近 4 周）

---

## 故障排除

### AI 功能不可用
- 检查 API Key 是否正确配置
- 点击"测试"验证 Key 是否有效
- 检查网络连接

### 笔记加载缓慢
- 检查笔记目录是否包含大量文件
- 使用排除文件夹功能排除不需要的目录

### 同步失败
- 检查 Obsidian Vault 路径是否正确
- 确认 Vault 中有 `.obsidian` 文件夹
