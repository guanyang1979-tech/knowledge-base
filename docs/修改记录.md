#### Bug 修复

1. **回收站功能修复** (`electron/main.ts`)
   - 问题：`get-trash-notes` 返回数组，但前端期望 `{ success, notes }` 对象
   - 修复：返回值改为 `{ success: true, notes }`

2. **批量导入功能修复** (`electron/main.ts`)
   - 问题：批量导入不支持 docx/xlsx/pdf 格式
   - 修复：复用 `import-document` 的完整解析逻辑，支持所有格式

3. **文件监控事件泄漏修复** (`electron/preload.ts` + `src/App.tsx`)
   - 问题：事件监听器没有清理机制
   - 修复：preload 暴露 cleanup 函数，App.tsx useEffect 中添加清理

4. **模板功能修复** (`src/components/Sidebar.tsx`)
   - 问题：模板点击创建假笔记，保存会失败
   - 修复：改为创建真实笔记并应用模板内容

5. **删除按钮可见性修复** (`src/components/NoteList.tsx`)
   - 问题：`group-hover` 无父 `group`，按钮不可见
   - 修复：在父容器添加 `group` 类

6. **SplashScreen 内存泄漏修复** (`src/components/SplashScreen.tsx`)
   - 问题：setTimeout 清理函数返回值未被使用
   - 修复：正确清理嵌套的 setTimeout

7. **自动保存竞态条件修复** (`src/components/NoteEditor.tsx`)
   - 问题：快速编辑时可能出现保存覆盖
   - 修复：使用 useRef 追踪最新内容和最后保存内容

8. **设置面板 CSS 冲突修复** (`src/components/SettingsModal.tsx`)
   - 问题：关闭按钮有重复的 dark mode 类名
   - 修复：清理重复的类名

9. **WelcomeDashboard DOM 查询修复** (`src/components/WelcomeDashboard.tsx`)
   - 问题：使用 `document.querySelector` 触发新建笔记，脆弱易失效
   - 修复：改用 props 回调 `onCreateNote`

#### 性能优化

1. **笔记列表加载优化** (`electron/main.ts`)
   - 优化前：每次读取所有笔记的完整内容
   - 优化后：只返回元数据（前 500 字节），内容按需加载
   - 预计提升加载速度 5-10 倍

2. **搜索性能优化** (`src/App.tsx`)
   - 添加 300ms 防抖，避免频繁触发状态更新

3. **日志写入优化** (`electron/main.ts`)
   - 使用内存缓冲，每 5 秒批量写入文件
   - 生产环境减少日志级别

#### 功能增强

1. **快捷键增强** (`electron/main.ts`)

4. **统计面板增强** (`src/components/StatisticsPanel.tsx`)

### UI 全面改版 — 暗色科技风格

**背景：** 用户反馈原有界面颜色太杂乱，要求统一为暗色科技风格，使用不同灰度的灰色和黑色组合，构建科技感和未来感的界面。

#### 改动文件

| 文件 | 类型 | 说明 |
|------|------|------|
| `src/components/SplashScreen.tsx` | 重写 | 启动封面改为纯黑灰暗色系，3秒后缩放+淡出消失 |
| `src/components/WelcomeDashboard.tsx` | 重写 | 欢迎仪表盘改为暗色科技风格 |
| `src/App.tsx` | 重写 | 主界面布局改为暗色统一风格 |
| `vite.config.ts` | 修改 | 添加 `base: './'` 修复 Electron 加载路径问题 |
| `package.json` | 修改 | 版本升级为 2.0.0，添加 author 字段 |

#### 设计变更

**启动封面：**
- 背景从彩色渐变改为纯黑灰渐变（`#0a0a0f` → `#0d0d14` → `#08080d`）
- 展示时长从 2.5 秒改为 3 秒
- 消失动画改为缩放(1.02) + 淡出
- Logo 从彩色渐变改为深色磨砂玻璃风格
- 外圈环从彩色改为极淡白色边框
- 进度条从彩色发光改为单像素白色细线
- 去除悬浮粒子等彩色装饰元素

**欢迎仪表盘：**
- 背景从彩色渐变改为暗色渐变，添加极淡网格纹理
- 统计卡片从彩色背景改为白色半透明边框 + 半透明背景
- 快捷操作按钮从彩色渐变改为极简边框按钮，hover 时微亮
- 最近笔记卡片统一为暗色风格
- 标签从灰色背景改为极简边框样式
- 小贴士区域改为低透明度边框 + 背景
- 文字颜色统一为白色 + 不同透明度等级（85% / 50% / 30% / 20% / 15%）

**主界面：**
- Header 从白色改为暗色半透明 + backdrop-blur 效果
- 搜索框、按钮图标全部改为白色低透明度
- 侧栏、笔记列表、AI 面板背景统一为 `#0d0d14`
- 所有边框统一为 `rgba(255,255,255,0.06)`
- 字体改为超细体（font-extralight）+ 大写字母间距（tracking-wider）

#### Bug 修复

- **Obsidian 同步计数为 0：** `syncedCount` 变量初始化后未递增，在文件同步成功后添加 `syncedCount++`
- **HTML 导出内容为 Markdown：** export-note 和 export-notes-batch 处理程序在 format=html 时只是复制 Markdown 内容，改为添加完整的 Markdown 到 HTML 转换逻辑
- **startup.bat 中文乱码：** CMD 无法解码 UTF-8 中文字符，改用纯 ASCII 重写
- **startup.bat 窗口不关闭：** 原 bat 有 `pause` 命令，改为 `exit`
