## 2026-04-10

### UI 全面改版 — 暗色科技风格

**背景：** 用户反馈原有界面颜色太杂乱，要求统一为暗色科技风格，使用不同灰度的灰色和黑色组合，构建科技感和未来感的界面。

#### 改动文件

| 文件 | 类型 | 说明 |
|------|------|------|
| `src/components/SplashScreen.tsx` | 重写 | 启动封面改为纯黑灰暗色系，3秒后缩放+淡出消失 |
| `src/components/WelcomeDashboard.tsx` | 重写 | 欢迎仪表盘改为暗色科技风格 |
| `src/App.tsx` | 重写 | 主界面布局改为暗色统一风格 |
| `vite.config.ts` | 修改 | 添加 `base: './'` 修复 Electron 加载路径问题 |
| `package.json` | 修改 | 版本升级为 2.0.0，添加 author 字段 |

#### 设计变更

**启动封面：**
- 背景从彩色渐变改为纯黑灰渐变（`#0a0a0f` → `#0d0d14` → `#08080d`）
- 展示时长从 2.5 秒改为 3 秒
- 消失动画改为缩放(1.02) + 淡出
- Logo 从彩色渐变改为深色磨砂玻璃风格
- 外圈环从彩色改为极淡白色边框
- 进度条从彩色发光改为单像素白色细线
- 去除悬浮粒子等彩色装饰元素

**欢迎仪表盘：**
- 背景从彩色渐变改为暗色渐变，添加极淡网格纹理
- 统计卡片从彩色背景改为白色半透明边框 + 半透明背景
- 快捷操作按钮从彩色渐变改为极简边框按钮，hover 时微亮
- 最近笔记卡片统一为暗色风格
- 标签从灰色背景改为极简边框样式
- 小贴士区域改为低透明度边框 + 背景
- 文字颜色统一为白色 + 不同透明度等级（85% / 50% / 30% / 20% / 15%）

**主界面：**
- Header 从白色改为暗色半透明 + backdrop-blur 效果
- 搜索框、按钮图标全部改为白色低透明度
- 侧栏、笔记列表、AI 面板背景统一为 `#0d0d14`
- 所有边框统一为 `rgba(255,255,255,0.06)`
- 字体改为超细体（font-extralight）+ 大写字母间距（tracking-wider）

**配色规范：**

| 用途 | 颜色 |
|------|------|
| 主背景 | `#0a0a0f` |
| 次背景 | `#0d0d14` |
| 最深背景 | `#08080d` |
| 边框 | `rgba(255,255,255,0.06)` |
| 边框 hover | `rgba(255,255,255,0.12~0.15)` |
| 文字主要 | `rgba(255,255,255,0.85~0.9)` |
| 文字次要 | `rgba(255,255,255,0.4~0.5)` |
| 文字弱化 | `rgba(255,255,255,0.15~0.3)` |
| 文字极弱 | `rgba(255,255,255,0.06~0.15)` |

## 2026-04-10

### 需求文档 v2.0 全面审计与修复

**背景：** 对照 `需求文档v2.0.md` 逐项审计项目，修复遗漏和半实现的功能。

#### 新增文件

| 文件 | 说明 |
|------|------|
| `scripts/dev.mjs` | 开发模式启动脚本（esbuild 构建主进程 + Vite 开发服务器 + Electron） |
| `startup.bat` | 纯 ASCII 启动脚本，自动发现 exe 并清除 `ELECTRON_RUN_AS_NODE` 环境变量 |
| `src/components/SplashScreen.tsx` | 启动封面（含项目名称、版本号 v2.0、作者 12792、渐变消失效果） |
| `src/components/WelcomeDashboard.tsx` | 欢迎仪表盘（统计卡片、快捷操作、最近笔记、小贴士） |

#### 修改文件

| 文件 | 说明 |
|------|------|
| `electron/main.ts` | 修复 sync-obsidian-notes 的 `syncedCount` 未递增问题；重写 export-note 和 export-notes-batch 的 HTML 导出逻辑（Markdown 转 HTML） |
| `src/App.tsx` | 集成 SplashScreen 和 WelcomeDashboard |
| `package.json` | 版本号 1.0.0 → 2.0.0，添加 author 字段 |

#### Bug 修复

- **Obsidian 同步计数为 0：** `syncedCount` 变量初始化后未递增，在文件同步成功后添加 `syncedCount++`
- **HTML 导出内容为 Markdown：** export-note 和 export-notes-batch 处理程序在 format=html 时只是复制 Markdown 内容，改为添加完整的 Markdown 到 HTML 转换逻辑
- **startup.bat 中文乱码：** CMD 无法解码 UTF-8 中文字符，改用纯 ASCII 重写
- **startup.bat 窗口不关闭：** 原 bat 有 `pause` 命令，改为 `exit`
